psi-plus theme: wonton soup
===========================

Chat theme for psi-plus+webkit based on the KDE 4 color scheme *wonton soup*. Theme is not entirely completed, just what I need currently.

## Requirements

* psi-plus + webkit (tested with 0.16.185-1)
* Font-Awesome font
